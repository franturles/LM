1. Descripcion del modulo

El modulo de PLE de DAI está orientado principalmente al arte de la Programacion. A lo largo del modulo se enseñan tecnicas de programacion tanto estructuradas como orientadas a objetos, trabajadas desde distintas
plataformas (Linux, Windows y dispositivos Moviles) y con diferentes finalidades; desde aplicaciones de gestion con acceso a base de datos hasta aplicaciones multimedia y juegos.
