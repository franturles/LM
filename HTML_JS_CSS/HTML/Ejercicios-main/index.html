Quienes somos

La chistera es un grupo de graciosos artistas dedicados al mundo de la magia,la comedia y el espectaculo. Entre nosotros podrás encontrar humoristas,magos,showmen, etc.

Tenemos multitud de espectaculos y numeros variados:

  1. Magia
    1. De Salon
    2. Ilusionismo
    3. Close up
  2. Chistes variados
  3. Parodias de personajes famosos
  4. Malabarismo
